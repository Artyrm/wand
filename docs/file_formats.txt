На данный момент используется два формата файлов:

Росчерки: название файла начинается с символа, который в нем записан (на данный момент - что-то из набора 'a', 'o', 's', 'x'), далее следует что-нибдь, обеспечивающее уникальность названия. Я использую unix-time, но эта информация в дальнейшем не используюется
Внутри - данные в формате numpy.savetxt по умолчанию:
float-числа, разденые пробелами. В каждой строке x, y, z координаты

Словарь: JSON-файл, содержит значение segmentation (сколько точека в каждой букве) и словарь эталонных букв. Буква записывается как список списков (результат numpy.ndarray.tolist), к трем координатам тут добавляется допустимая дисперсия.
