Это, на самом деле, надо будет занести в докстринги выживших модулей. После чего файл можно будет спокойно удалить.

Selector.py - модуль, который хранит в себе словарь, и говорит, на какие буквы похож росчерк. По сути - обертка к unify_definition
Stroke.py - выделяет штрих из последовательности данных
StrokeViewer.py - программка для отсмотра и ручной классификации штрихов. Позволяет посмотреть, как он выглядит и назначить букву
letters/ - штрихи, распознанные вручную
learned/ - автоматически распознанные штрихи, классифициованные соответствующим образом
make_core.py - скрипт, собирающий распознанные штрихи в общее описание. В будущем должне объединиться с StrokeViewer'ом
stroke_widget.py - базовый виджет для отображения штрихов на основе PyQtGraph
wand_gui.py - основной файл
